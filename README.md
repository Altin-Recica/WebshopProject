# Altin's T-Shirt Webshop

This is a study project for **KDG University College**, in which I created a T-shirt webshop to showcase my web development skills. The project is built with **HTML, CSS, and JavaScript**.

## Features

* Product overview and detail pages
* Interactive shopping cart (add/remove items)
* Visually appealing layout with CSS
* Shop with search and filter functionality

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Altin-Recica/WebshopProjectKDG.git
```

2. Open `index.html` in your browser.

## Technologies

* HTML
* CSS
* JavaScript
